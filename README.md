# primerProjecte
En el programa exercici1.js trobarem els exercicis del Full de Treball 1 de la Pla 3.
1) Donada una cadena de caràcters definida dins del programa, que el programa ens compti quantes lletres "a" hi ha dins la cadena.
2) Aconseguir que el programa ens digui la lletra introduïda per l'usuari pel teclat.
3) Fer que l'usuari ens doni la paraula i la lletra a buscar.
4) i 5) Donada una cadena de caràcters que el programa digui la posició de la primera "a" o lletra demanada per l'usuari. En aquest cas m'he saltat l'exercici 4 per pàssar directament al 5.
6) Fer un programa que guardi en una llista les posicions de totes les lletres "a" que es trobin en la cadena.
7) Donada una cadena numèrica, amb números del 0 al 9, comptar el nombre de vegades que apareix cada número i guardar-ho en una llista
8) Utilitzant l'exercici 6, si a la cadena hi ha algún element numèric que el programa avisi.
